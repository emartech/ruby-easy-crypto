# EasyCrypto

Provides simple wrappers around the openssl crypto implementation. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'easy-crypto'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install easy-crypto
